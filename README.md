ratel
=====

xcos to HDL toolbox

Allows the graphical design of digital circuitry targeted at signal
processing applications on FPGAs. 

Intended to be; 
* vendor-agnostic
* easy to modify, debug, extend
* non-restrictive in terms of cost or access
